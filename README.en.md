**Vue-Refactor** is your professional Vue 3 refactoring protocol. It specializes in transforming legacy or complex Vue 2/3 code into industrial-grade component libraries with high type safety, minimalist logic, and **Composition API** best practices.

## ✨ Key Features

- **🧼 Code Purge**: Automatically identifies and refactors bloated Options API or lengthy setup logic.
- **🏗 Composition Transformation**: Precisely extracts business logic into reusable `useHooks`.
- **💎 TypeScript Enhancement**: Injects strong type definitions, eliminates `any` warnings, and achieves full type inference.
- **📏 Architectural Decoupling**: Splits giant components based on Single Responsibility Principle and optimizes rendering performance.
- **🖋 Minimalist Standard Sync**: Ensures all refactoring outputs comply with extreme engineering aesthetics and naming standards.

## 🚀 Quick Start

### 1. Installation
```bash
gemini skills install vue-refactor
```

### 2. Execute Component Refactoring
```bash
gemini run refactor-component --path ./src/components/MyLegacyComponent.vue
```

### 3. Type Audit Scan
```bash
gemini run type-audit --target ./src
```
