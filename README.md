**Vue-Refactor** 是你的 Vue 3 极致重构专家协议。它专注于将 Vue 2/3 的陈旧或面条代码重构为具备高度类型安全、极简逻辑以及符合 **Composition API** 最佳实践的工业级组件库。

## ✨ 核心特性

- **🧼 代码净化 (Code Purge)**：自动识别并重构臃肿的 Options API 或冗长的 setup 逻辑。
- **🏗 Composition 转换**：将业务逻辑精准抽离为可复用的 `useHooks`。
- **💎 TypeScript 增强**：自动注入强类型定义，消除项目中的 `any` 警告，实现完全的类型推导。
- **📏 架构解耦**：按照单一职责原则，自动拆分巨型组件，优化渲染性能。
- **🖋 极简规范同步**：确保所有重构产出均符合极致的工程审美与命名规范。

## 🚀 快速开始

### 1. 安装
```bash
gemini skills install vue-refactor
```

### 2. 执行组件重构
```bash
gemini run refactor-component --path ./src/components/MyLegacyComponent.vue
```

### 3. 类型增强扫描
```bash
gemini run type-audit --target ./src
```
